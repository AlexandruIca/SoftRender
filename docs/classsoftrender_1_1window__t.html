softrender::window_t class

Use this to construct a window.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

This is just a wrapper around SDL to construct a surface from memory and blit it.

Use SOFTRENDER_MOCKING if you want to disable drawing to the screen.
